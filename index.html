Instalación

Android 4.4 o superior.

  • Descargar el .apk en el telefono o moverlo desde la pc.
  • Abrir el archivo .apk para instalar.
  • Permitir fuentes/origen desconocidas(Se puede desactivar luego de la instalación).

Uso

El boton flotante permite comenzar un nuevo metodo de entrenamiento. El historial muestra los metodos que ya han sido completados.

Al crear un metodo, el RM/6RM/3RM se usará para calcular los pesos correspondientes.

Un metodo pasa al historial cuando todos sus ciclos son marcados como realizados, a excepción del test que lo hará de forma automatica.

Por ejemplo, para comenzar un metodo ERPA:

  • Seleccionar el ejercicio correspondiente en la pantalla principal.
  • Tocar el boton flotante, seleccionar ERPA e ingresar el valor del 6RM/3RM con el que se desea trabajar.
  • Al finalizar marcarlo como hecho.
  • En caso de haber realizado un ajuste, es recomendable crear un nuevo ERPA con el valor ajustado, quedando este listo para la proxima clase ;)
